# Mission Possible OS

#**Overview**

Mission Possible OS is a NASA-inspired WebOS created for the Stardance challenge. It is a browser-based desktop environment that combines space exploration themes with interactive applications and games.

#**Features**
1. Space-themed desktop interface
2. Custom pixel-art inspired design
3. Interactive homepage
4. NASA resource integration
5. Planned Features
6. Mission Log application
7. Calculator
8. Digital Clock
9. Flappy Rocket game
10. Whack an Alien game
11. Snake game
12. Draggable desktop windows
13. Additional astronaut-themed challenges

#**Technologies Used**
HTML
CSS
JavaScript
GitHub Codespaces
AI Usage

#**Future Goals**

The project aims to evolve into a complete educational WebOS that encourages creativity, coding, and interest in space exploration.
